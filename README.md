Welcome to **Zeforc**. We are a minimalist software lab dedicated to developing high-performance, privacy-focused utilities and core libraries. Our tools are meticulously crafted to bridge the gap between human thought and digital expression.

## 📐 Core Philosophy

Everything built at Zeforc is rooted in a three-pillar philosophy, guiding our engineering and design choices:

- **Zen (The Mindset):** Absolute focus and clarity. We strip away the unnecessary, favoring clean architectures and minimalist user experiences.
- **Forge (The Process):** Relentless iteration and craftsmanship. Writing robust, efficient, and reliable code from the ground up.
- **Arc (The Structure):** Seamless connectivity. Building elegant bridges across different platforms and ecosystems.

## 🚀 Current Focus

Our primary mission right now is building precise, high-performance tools that empower developers and enrich user experiences:

- **[DeboxHub](https://deboxhub.com):** A minimalist, lightning-fast, and 100% privacy-focused developer toolkit. Featuring a comprehensive suite of essential, client-side utilities engineered for speed and security.
- **[LichTa](https://www.npmjs.com/package/lichta):** An open-source, high-precision npm library for Gregorian ↔ Vietnamese Lunisolar calendar conversion. Built with a pure core architecture to seamlessly support modern frameworks (React, Vue, Svelte, Angular).

## 🛠 Tech Stack

We believe in using the right tools to forge the best solutions. Our current development stack revolves around:

- **Ecosystem:** `TypeScript` / `JavaScript` (Node.js & Browser environments).
- **Architecture:** Pure core logic separation, multi-framework integration, and strict client-side processing for privacy.
- **Design:** Geometric precision, purely functional UI/UX.
